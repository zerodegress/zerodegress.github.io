
# 冰点启航的个人主页  

嗨，这是冰点启航的个人主页，现在目前为止还没有太多的内容，但是很快这里将会得到更新。  

这个网页的主人正在研究Rust编程语言，也许以后会发布一些相关内容。  
同时，HTML和CSS的前端学习也在日程表上，也许以后可以将这个网页更新为更漂亮的形式。  
  
这个网页的主人是[[Rusted Wiki|<https://rustedwarfare.org>]]的[[编辑者|<https://rustedwarfare.org/wiki/%E7%94%A8%E6%88%B7:%E5%86%B0%E7%82%B9%E5%90%AF%E8%88%AA>]]，致力于收集整理Rusted Warfare的相关资料，并为Wiki提供更多模板。  
  
这个网页的主人是[[MC百科|<https://mcmod.cn>]]的一名编辑员，同时也是[[CFPAOrg|<https://github.com/CFPAOrg/Minecraft-Mod-Language-Package>]]项目的参与者，致力于收集·整理·翻译Minecraft的大小模组。
